# Lengua de Señas App

**Lengua de Señas App** es una plataforma educativa diseñada para facilitar el aprendizaje y la gestión de recursos para la enseñanza de la lengua de señas. Permite a docentes y estudiantes interactuar con material educativo de manera sencilla e intuitiva.

---

##  Funcionalidades principales

###  Docentes:
- Crear, editar y eliminar recursos didácticos (PDFs, enlaces, etc.).
- Visualizar un dashboard con estadísticas de sus recursos.
- Gestionar su perfil y seleccionar un avatar.

###  Estudiantes:
- Iniciar sesión y gestionar perfil personal.
- Elegir y guardar un avatar personalizado.
- Acceder a recursos educativos de forma organizada.

###  Administración:
- Panel de administración para gestionar usuarios y docentes.
- Visualización de estadísticas generales: cantidad de usuarios, docentes y recursos.

---

##  Tecnologías utilizadas
- **Backend:** ASP.NET Core Web API (C#)  
- **Frontend:** HTML, CSS y JavaScript  
- **Base de datos:** SQL Server  
- **Control de versiones:** Git & GitHub  
- **Persistencia en el cliente:** LocalStorage (para mantener sesión y avatar)
